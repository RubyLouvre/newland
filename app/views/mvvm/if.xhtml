if绑定

if绑定属性于流程绑定的一种，也是模板绑定的一种。当它的值为一个监控者或函数数时， 则执行它，取其返回值，否则就直接像JS代码那样进行布尔转换。 如果为真，则渲染此元素的内容，为假则就此结束，元素的innerHTML为空。

ZZZZZZZZZZZZZZZZZZZZZZZZZZZZZ

HTML：

    <p bind="if: show">
       XXXXXXXXXXXXXXXXXXXXXXXXXXXX
    </p>
    <p bind="if: function(){return 1}">
       YYYYYYYYYYYYYYYYYYYYYYYYYYYYY
    </p>
    <p bind="if:2">
       ZZZZZZZZZZZZZZZZZZZZZZZZZZZZZ
    </p>

JS。

    $.require("ready,./more/avalon", function(){
        var model = $.ViewModel({
            show: false
        });
        $.View(model);
        setTimeout(function(){
            model.show(true)
        },2000);
    })
