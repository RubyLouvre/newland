text绑定

第一个传参将强制转换为字符串，假值为""，然后变成目标节点的innerText!

示例

xxxxxxxxxxxxx

HTML：

   <p bind="text: content" /> xxxxxxxxxxxxx </p>

JS：

    $.require("ready,./more/avalon", function(){
        var model = $.ViewModel({
            content: "这是来自model的"
        });

        setTimeout(function(){
            $.View(model);
        },1000)

    })
示例2 这东西 。

HTML

   这东西 <span bind="text: priceRating"></span>。

JS

     $.require("ready,./more/avalon", function(){
            var model = $.ViewModel({
                price: 24.95,
                priceRating: function(){
                    return this.price() > 50 ? "有点贵" : "比较便宜";
                }
            });
             $.View(model);
        });
