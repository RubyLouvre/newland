with绑定

with绑定的参数对应ViewModel中的一个对象。

示例

Latitude: , Longitude:

HTML：

    <h1 bind="text: city"> </h1>
    <p bind="with: coords">
        Latitude: <span bind="text: latitude"> </span>,
        Longitude: <span bind="text: longitude"> </span>
    </p>

JS:

    $.require("ready,./more/avalon", function(){
        var model = $.ViewModel( {
            city: "London",
            coords: {
                latitude:  51.5001524,
                longitude: -0.1262362
            }
        });
        $.View(model);
    })
