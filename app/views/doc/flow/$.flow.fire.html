$.flow.fire(name,args1,args2,...)

描述：

通知操作流，这个操作已经完成，将参数传递给准备触发的最后回调。

参数：

name
必须。字符串。单个操作名。你也可以当成是一种自定义事件。
args
可选，个数不受限制
例子 
$.require("flow",function(){
    var flow = new $.Flow;
    flow.bind("aaa,bbb", function(){
        $.log(Array.apply([],arguments),true)
    });
    flow.fire("aaa",1,2,3,4)
    flow.fire("bbb",333);//到这里上面的回调打印1,2,3,4,333
    flow.fire("aaa",10,20)//到这里上面的回调打印10,20,333
})
