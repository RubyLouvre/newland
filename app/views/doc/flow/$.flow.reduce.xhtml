$.flow.append(names,name)

描述：

在一个回调执行前追减少一个要执行的操作步骤。

参数：

names
必须。如果是字符串,则相当于$.flow.find中的操作名列表，如果是对象，则相等于$.flow.find(opts)，可以添加更多的筛选条件。
name
必须。字符串。要免去的操作名。
例子 

$.require("flow", function(){
    var node = new $.Flow();
    node.bind("aaa,bbb,ccc", function(){
        $.log(Array.apply([],arguments),true)
    });

    node.fire("aaa",1);
    node.reduce("aaa","ccc")
    node.fire("bbb",2);//立即执行回调
    node.fire("ccc",3);//此句没存在的价值了
})
