$.parseData(elem, name, id, table)

描述：

取得元素的某一属性，转换为JSON数据。此API是框架用于转换HTML5的data-*属性的。

转换规则：

  • "true", "false", "null" 转换true,false,null
  • 形如数字的字符串转为数字
  • 如果是用花括号包起来的尝试转换为对象
  • 其他原样输出

参数：

elem
必需。元素节点。
name
必需。String。属性名。
id
可选。Number。元素的UUID。
table
可选。Object。将转化后的属性值保存到一个对象之上。

返回值：

经转换后的数据。

例子 
            <div id="test" data-obj="{a:'a',b:'b',c:'c'}" data-arr="[1,2,3,4]" data-nan="NaN" data-str="aaa"></div>
    
       $.require("ready,data",function(){
                var el = $("#test")[0]
                var data = $.parseData(el,"data-obj");
                $.log($.type( data ),true);
                $.log($.dump(data),true);
                var array = $.parseData(el,"data-arr");
                $.log($.type( array ),true);
                $.log($.dump(array),true);
                var nan = $.parseData(el,"data-nan");
                $.log($.type( nan ),true);
                $.log($.dump( nan ),true);
                 var str = $.parseData(el,"data-str");
                $.log($.type( str ),true);
                $.log($.dump( str ),true);
       });
