$.fn.slice(start,end)

描述：

对原实例进行切片操作，行为与数组的slice相似，生成新的mass实例返回。

参数：

start
可选。要抽取的片断的起始下标。如果是负数，从后面取起。
end
可选。规定从何处结束选取。

返回值：

新的mass实例

例子 
            $.require("ready,node",function(){
               var a = $("article *");
               $.log(a.length,true);
               $.log(a.slice(4).length,true)
            });
