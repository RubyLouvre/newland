$.fn.html(val)

描述：

采用get one set all方法操作节点的innerHTML属性。也就是说，如果没有参数，则返回第一个元素节点的innerHTML，否则给所有匹配元素节点设置新 的innerHTML，然后返回实例本身。

参数：

val
可选。String。
例子 
      $.require("ready,node",function(){
          $(".test").html('<span style="color:red">司徒正美</span>')
       });
