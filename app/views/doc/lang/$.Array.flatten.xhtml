$.Array.flatten()

描述：

返回一个不具有嵌套层次结构的数组，内嵌的数组递归加入到最外层数组中。

例子 
                $.require("lang",function(){
                    var ret = $.Array.flatten([1, [2, [3, 4, 5] ],6]);
                    ret = $.dump(ret);//序列化
                    $.log(ret,true);
                });
