$.parseJSON(code)

描述：

将符合JSON格式的字符串转换为JSON对象。

code
必需。String。
例子 
                $.require("lang",function(){
                    var a = '{"name":"John"}';
                    a = $.parseJSON(a);
                    $.log(a.name === "John",true);
                    $.log($.isPlainObject(a),true);
                });
