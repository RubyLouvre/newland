$.String.escapeRegExp( [str | reg] )

描述：

将目标字符串中可能会影响正则表达式构造的字符串进行转义。

静态形式下,第一个参数可以是正则.

例子 
                $.require("lang",function(){
                    var a = $.lang(".animals.sheep[1]");
                    $.log(a.escapeRegExp(), true);
                });
