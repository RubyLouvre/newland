$.Array.pluck( arr, name )

描述：

获取所有元素的同一个属性的值，组成新数组返回。

name
必需。String。元素的某一属性名。
例子 
                $.require("lang",function(){
                    var arr = [{name : 'moe', age : 40}, {name : 'larry', age : 50}, {name : 'curly', age : 60}]
                    var ret = a.pluck("name")
                    $.log($.dump( ret ),true);
                });
