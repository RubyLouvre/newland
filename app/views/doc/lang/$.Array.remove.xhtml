$.Array.remove( arr, el )

描述：

破坏性方法，修改原对象。移除第一个匹配的元素。

参数：

el
必须。要移除的元素。

返回值：

返回布尔值表示是否移除成功与否。

例子 
                $.require("lang",function(){
                    var test = [3, 5, 6, 1, 20];
                    var a = $.lang(test);
                    $.log(a.remove(5),true);
                    $.log($.dump(test),true);
                });
