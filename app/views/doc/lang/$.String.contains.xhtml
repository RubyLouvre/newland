$.String.contains(str)

描述：

判断目标字符串是否包含str这个子串。

参数：

  • str必需。String。
例子 
                $.require("lang",function(){
                    var a = $.lang("司徒正美司徒正美");
                    $.log(a.contains("正美"),true);
                });
