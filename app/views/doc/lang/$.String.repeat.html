$.String.repeat( str, n )

描述：

将目标字符串重复n次，构成新的字符串返回。

参数：

  • n必需。Number。重复次数。
例子 
                $.require("lang",function(){
                    var a = $.lang("司徒正美");
                    $.log(a.times(4),true);
                });
