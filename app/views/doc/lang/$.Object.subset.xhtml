$.Object.subset( obj, arr )

描述：

根据传入数组取当前对象相关的键值对组成一个新对象返回。

参数：

arr
必需。新对象的键名。
例子 
                $.require("lang",function(){
                    var obj = {
                        a:"one",
                        b:"two",
                        c:"three"
                    };
                    var ret = $.Object.subset( obj,["a","b"] );
                    $.log( $.dump(ret), true);
                });
