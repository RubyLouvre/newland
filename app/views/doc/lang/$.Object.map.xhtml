$.Object.map( obj, callback, scope )

描述：

取得对象的键值对，依次放进回调中执行,并收集其结果，以数组形式返回。

参数：

callback
必需。回调函数，它的参数依次是对象的某个键对应的值，键名
scope
可选。回调函数里面的this值。
例子 
                $.require("lang",function(){
                    var obj = {
                        a: 0,
                        b: 1,
                        c: 2
                    };
                    var arr  = $.Object.map( obj, function( v, k){
                         return v + 10
                    });
                    $.log( $.dump(arr), true );
                });
