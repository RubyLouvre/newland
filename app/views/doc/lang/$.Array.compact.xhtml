$.Array.compact(arr)

描述：

返回一个不包含null与undefined的副本。

例子 
                $.require("lang",function(){
                    var arr = [3, 5,null, 6, undefined,1]
                    var ret = $.lang(arr).compact()
                    ret = $.dump(ret);//序列化
                    $.log(ret,true);
                });
