array.union(array)

描述：

求目标数组与array的并集。

array
必需。Array。
例子 
                $.require("lang",function(){
                    var a = [1,2,3];
                    a = $.lang(a).union([2,3,4,5,6]);//取并集
                    $.log(a,true)//[1,2,3,4,5,6]
                });
