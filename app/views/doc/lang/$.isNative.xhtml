$.isNative(host,method)

描述：

判定method是否为host对象的原生方法，用于向前兼容，详细讨论请看这里。

host
必需。Object。要检测的对象。
method
必需。String。要检测的方法名。

返回值：

Boolean

例子 
                $.require("lang",function(){
                    $.log( $.isNative(window,"alert"),true);
                    window.test = {
                        toString: function() {
                            return '[function]';
                        }
                    };
                    $.log( $.isNative(window, 'test'),true);
                });
