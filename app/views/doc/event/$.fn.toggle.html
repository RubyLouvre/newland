$.fn.toggle( fn, fn1 ,[ fn2 ],[ fnn ])

描述：

绑定两个或多个单击事件，以便让每次点击时的回调都不一样。比如，我们绑定了四个函数，那么第五次点击它时，它执行第一个回调，这样周而复始的重复执行。

参数：

fn
必需。Funcion。第一数次点击时要执行的函数。
fn1
必需。Funcion。第二数次点击时要执行的函数。
fn2
可选。Funcion。第三数次点击时要执行的函数。
fnn
可选。Funcion。第N数次点击时要执行的函数。
例子

请单击这里

            $.require("ready,event",function(){
                function log(el,text){
                    $(el).append("执行第"+text+"个回调函数
");
                }
                $(".sample1").toggle(function(){
                    log(this,1)
                },
                function(){
                    log(this,2)
                },
                function(){
                    log(this,3)
                } )
            })
