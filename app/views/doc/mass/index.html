模块加载模块

描述：

mass Framework的最核心的种子模块，提供构筑框架的最基本工具， 包括模块加载系统，mix机制， 数组化，类型判定，别名机制，调试机制，domReady......

主要API介绍

API 解说
require 请求一个或N个模块，当它们都加载本地时，执行回调函数（第二个参数），如果有一个没加载回来， 执行第三个参数（错误回调，可选）
define 定义一个模块，它有三个参数，第一个为模块名，第二个是依赖列表（可选），第三个是模块本身（为一个函数）
mix 将一个或多个对象合并到第一个参数（它也必须是对象）中， 如果只有一个参数，则合并到mix的调用者上，如果最后一个参数是布尔，则用于判定是否覆盖已有属性
slice 将类数组对象转换成真正的数组，并进行切片操作(如果第二第三参数存在的情况下)
type 用于取得数据的类型（一个参数的情况下）或判定数据的类型（两个参数的情况下）
log 打印调试信息到控制台(1个参数的情况)或页面底部(第二参数为true或1的情况下)
oneObject 创建一个对象，其键值都为1(如果没有指定)或第二个参数，用于用于高速化判定
bind 简单的事件绑定
unbind 简单的事件卸载
noop 一个空函数
html 返回HTML节点
head 返回HEAD节点
rword 一个有用的正则，将字符串以空白或“,”切割成一个个单词
mass 用于储存框架当前的版本号
exports 将内部的对象释放到全局作用域下
